# test-node-repo

## 豪华版影视订阅源

### 推荐入口

| 类别 | 文件 | 主要功能 |
|------|------|----------|
| 主订阅 | `sub.txt` | 整合稳定精选 / 采集聚合 / 备用线路 |
| 单接口 | `m.json` | 单文件 JSON 接口备用 |

### 推荐链接

| 类别 | 工具 | 主要功能 |
|------|------|----------|
| 稳定订阅 | ghproxy | `https://ghproxy.net/https://raw.githubusercontent.com/yunhugy/test-node-repo/main/sub.txt` |
| 单接口 | jsDelivr | `https://cdn.jsdelivr.net/gh/yunhugy/test-node-repo@main/m.json` |
| 仓库 | GitHub | `https://github.com/yunhugy/test-node-repo` |

### 订阅结构
- 稳定精选：优先推荐日常使用
- 采集/聚合：源更多，适合扩展
- 备用：当主线路不稳定时切换
