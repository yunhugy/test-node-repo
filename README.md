# test-node-repo

## 扩充版影视订阅源

### 推荐地址

| 文件 | 用途 | 推荐链接 |
|------|------|----------|
| `sub.txt` | TVBox/影视仓多线路订阅 | `https://ghproxy.net/https://raw.githubusercontent.com/yunhugy/test-node-repo/main/sub.txt` |
| `m.json` | TVBox/影视仓单线路接口 | `https://cdn.jsdelivr.net/gh/yunhugy/test-node-repo@main/m.json` |

### 说明
- `sub.txt` 已扩充为多条常见公开订阅
- 若个别源失效属于上游问题，可继续替换维护
- 你的当前网络更适合使用 ghproxy / jsDelivr 加速链接
