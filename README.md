# test-node-repo

## 影视订阅源（静态部署）

### 可用地址

| 文件 | 用途 | Raw 链接 |
|------|------|----------|
| `sub.txt` | TVBox/影视仓多线路订阅 | `https://raw.githubusercontent.com/yunhugy/test-node-repo/main/sub.txt` |
| `m.json` | TVBox/影视仓单线路接口 | `https://raw.githubusercontent.com/yunhugy/test-node-repo/main/m.json` |

### 使用方式
1. 复制上面的 Raw 链接
2. 粘贴到 TVBox / 影视仓 / 小苹果等 App 的订阅地址栏
3. 保存即可使用
